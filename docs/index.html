Speed Limit

Privacy Policy — Last updated: March 13, 2026

Overview

Speed Limit ("the App") displays the current speed limit as a floating overlay on your Android device. This privacy policy explains what data the App accesses and how it is used.

Location Data

The App uses your device's GPS to determine your current location. This location data is used solely to look up the speed limit of the road you are on.

  • Your location is never stored on-device or on any server.
  • Your location is never shared with any third party for analytics, advertising, or any other purpose.
  • Location data is sent only to the OpenStreetMap Overpass API in the form of anonymous latitude/longitude queries to retrieve road speed limit data. These queries contain no identifying information.

Background Location

The App requests background location access so that the floating speed limit overlay can continue to update when the App is not in the foreground or the screen is locked. Background location data is used identically to foreground location data — only for speed limit lookups, never stored or shared.

Data Collection

The App does not collect, store, or transmit any personal data. Specifically:

  • No user accounts or sign-in
  • No analytics or crash reporting
  • No advertising or ad tracking
  • No cookies or local storage of personal information
  • No contact, camera, microphone, or file access

Third-Party Services

The App communicates with the following third-party service:

  • OpenStreetMap Overpass API — Used to query road speed limit data. Queries contain only geographic coordinates (latitude and longitude) and no personally identifiable information. See the OpenStreetMap Foundation Privacy Policy.

Permissions

The App requests the following Android permissions:

  • Fine Location / Background Location — To determine which road you are on and display the speed limit.
  • Display Over Other Apps — To show the floating speed limit bubble over other applications.
  • Foreground Service — To keep the speed limit overlay running when the App is in the background.
  • Internet — To query the Overpass API for speed limit data.
  • Wake Lock — To keep GPS tracking active when the screen is off.

Children's Privacy

The App is not directed at children under 13 and does not knowingly collect any personal information from children.
